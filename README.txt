== Description ==
CRM Pro is a plugin designed to help you connect your favorite CRM platforms to WordPress.

**CRM Pro Features**

* Connects to your favorite CRM platforms
* Enables access to connect to live chat platforms
* Enables access to connect to forms platforms

Public support requests or issues with the plugin can be reported via the support tab of this listing or by emailing CRM Pro directly at help@crmproplugin.com

Review our terms of use and privacy policy at the following links: https://crmproplugin.com/terms-of-use/ and https://crmproplugin.com/privacy/

== Installation ==
1. Install CRM Pro either via the WordPress.org plugin directory, or by uploading the .zip file to your server and 'Activate' it.
2. Go to the CRM Pro plugin API settings page in the admin menu on the left and connect your favorite CRM
